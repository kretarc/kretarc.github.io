# krestic-portfolio

Personal portfolio website for **Krestic** — 15-year-old developer & AI builder from Dhanbad, India.

🌐 **Live:** [kemicarese.github.io](https://kemicarese.github.io) *(or wherever you host it)*

---

## About

A clean, dark editorial portfolio built as a single HTML file. No frameworks, no build step, no dependencies — just drop it and it works.

Designed to showcase projects, stack, and contact info in a minimal but visually sharp aesthetic.

---

## Features

- Single-file HTML — zero dependencies, zero build tools
- Dark mode editorial design with DM Serif Display + DM Mono typography
- Custom dot cursor with hover expand effect
- Scroll-reveal animations on all sections
- Fully responsive (mobile-friendly)
- Smooth scroll navigation

## Tech Used (Portfolio itself)

- Vanilla HTML/CSS/JS
- [DM Serif Display](https://fonts.google.com/specimen/DM+Serif+Display) + [DM Mono](https://fonts.google.com/specimen/DM+Mono) + [Instrument Sans](https://fonts.google.com/specimen/Instrument+Sans) via Google Fonts
- IntersectionObserver API for scroll reveals
- No frameworks, no npm, no bundler

---

## Deploy

Works on any static host:

**GitHub Pages** — push to `gh-pages` branch or root of repo, enable Pages in settings.

**Netlify / Vercel** — drag and drop the `index.html` file (rename from `krestic-portfolio.html`).
